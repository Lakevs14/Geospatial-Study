Snow Water Equivalent & Temperature Dynamics in Colorado Counties

A longitudinal analysis of annual SWE trends across San Juan, Gunnison, and Summit counties from 1980 to 2025, integrating PRISM climate data, SNOTEL station records, and SVM-based Landsat classification.

1980 – 2025 88 weather stations 3 counties PRISM, SNOTEL & Landsat

At a glance

Study overview

This study investigates the relationship between annual average temperature and snow water equivalent across three high-elevation Colorado counties. Using spatially-averaged SNOTEL station data and PRISM-derived temperature rasters, we quantify the strength of temperature–SWE correlations and model long-term trends through linear regression. A support vector machine (SVM) classifier applied to Landsat imagery extends the analysis to land cover change detection at Lake Powell between 1985 and 2025.

SVM classification

Land cover classification — Landsat imagery

A support vector machine (SVM) with an RBF kernel was trained on region-of-interest (ROI) samples drawn from Landsat scenes for both 1985 and 2025. Six spectral bands were used as features, normalized per band using z-score standardization. Training samples were balanced to 5,000 pixels per class. The model was evaluated on a stratified 80/20 hold-out test set (n = 4,000 pixels).

Classification pipeline

Step 01
Image loading & normalization
Landsat scenes loaded via rasterio. Per-band z-score normalization applied globally across each image.
Step 02
ROI sampling
Training pixels extracted from shapefile ROIs. Up to 20,000 pixels per ROI, balanced to 5,000 per class with replacement if needed.
Step 03
SVM training
SVC with RBF kernel, C=10, gamma='scale', class_weight='balanced'. Trained on pooled 1985 + 2025 samples.
Step 04
Tile-based prediction
Full rasters classified in 256×256 pixel tiles to manage memory. Output saved as single-band uint8 GeoTIFF.
Step 05
Accuracy assessment
OA, AA, Cohen's kappa, confusion matrix, and per-class precision/recall/F1 reported on the held-out test set.

Methodology

Data & methods

  • 1 SWE data — Annual SWE averages computed from NRCS SNOTEL stations within each county boundary. Station values averaged to produce a single county-level SWE estimate per year.
  • 2 Temperature data — Annual average temperatures derived from PRISM 4km gridded rasters, clipped to county boundaries in GIS. Zonal statistics (mean) extracted per year.
  • 3 Statistical analysis — Pearson correlation coefficients computed between annual temperature and SWE for each county. Linear regression models fitted and visualized using Python (scipy, matplotlib).
  • 4 SVM classification — Support vector machine with RBF kernel (C=10, gamma='scale') trained on balanced ROI samples from Landsat 1985 and 2025 scenes. Z-score normalized spectral features across 6 bands. Tile-based prediction (256×256) applied to full scenes.
  • 5 Change detection — Binary water masks derived from SVM outputs compared pixel-by-pixel. Water area change quantified in km² accounting for raster CRS and pixel resolution.
  • 6 Significance testing — Two-tailed p-values reported for each correlation. Significance threshold set at α = 0.05. SVM accuracy reported via OA, AA, Cohen's kappa, and per-class F1 scores.
